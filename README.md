# 泛型
- 优点 
    * 减少相似代码
    * 通过抽象，增加拓展性
    * 常用于结构体、枚举、函数签名
    * 多个参数时，trait bound 可以保证对应的类型时一致的
- 缺点 
    * 过多的泛型，降低代码可读性
- 注意点 
   * 编译时，泛型会被具体的类型替代，但是这不影响执行效率
   * 函数签名使用了过多的泛型，代码需要优化，提高可读性和可维护性


# 生命周期
- 一般变量是在它声明的大括号`{}`里就是它的生命周期
```rust
{
    let a = 1; 
    // ...
    // end
}
```
- 一个借用的变量在借用之前或者引用之前，就是它的生命周期

```rust
{
    let a = "123".to_string();
    let b = a; // end a 的生命周期
    // 这里之后不可使用a 变量
}
```
